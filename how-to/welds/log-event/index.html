    Log event

    Each weld can have different events logged. The ability to log an event depends on the user's role and the current status of the weld. Here are some examples:

    • Welding events can only be logged for welds that have not been welded yet or have been rejected by an NDT operator. Welding events can only be logged by welders.
    • Heat treatment events can only be logged for welds that have already been welded. Heat treatments can only be logged by a heat treatment operator.
    • NDT inspection events can only be logged for welds that have been welded and heat treated (if required). NDT inspections can only be logged by an NDT operator.

    Admin users have the ability to log events on behalf of other users.

    Log welding event

    To log a welding event, you need to meet the following prerequisites:

    • Participating in the project as a welder or having admin status.
    • Weld has a status that accepts a welding event (does not have other events logged such as welding or heat treatment events, but can be rejected by an NDT operator).

    To log a welding event, follow these steps:

    1. Navigate to the details page of the weld that you want to log the event.
    2. If prerequisites are met, the weld will have a button "Log weld" (A). Click on the button.
    3. Choose the time of the event (B), and if you are an admin, choose the welder from the dropdown list (C). If you are a basic user, your name is selected by default.
    4. When you have filled in the time (and the welder), click on the "Log" button (D) to create the new welding event.

    By following these steps, you can create a new welding event for your weld. Logging of events helps ensure that the welding process is well-documented and traceable.

    Log heat treatment event

    To log a heat treatment event, you need to meet the following prerequisites:

    • Participating in the project as a heat treatment operator or having admin status.
    • Weld has a status that accepts a heat treatment event (has welding event logged).

    To log a heat treatment event, follow these steps:

    1. Navigate to the details page of the weld that you want to log the event.
    2. If prerequisites are met, the weld will have a button "Log heat treatment" (A). Click on the button.
    3. Choose the time of the event (B), and if you are an admin, choose the heat treatment operator from the dropdown list (C). If you are a basic user, your name is selected by default.
    4. When you have filled in the time (and the heat treatment operator), click on the "Log" button (D) to create the new heat treatment event.

    By following these steps, you can create a new heat treatment event for your weld. Logging of events helps ensure that the welding process is well-documented and traceable.

    Log NDT inspection event

    To log an NDT inspection event, you need to meet the following prerequisites:

    • Participating in the project as an NDT operator or having admin status.
    • Weld has a status that accepts an NDT inspection event (has welding event logged and heat treatment event logged (if required)).

    To log an NDT inspection event, follow these steps:

    1. Navigate to the details page of the weld that you want to log the event.
    2. If prerequisites are met, the weld will have a button "Log NDT" (A). Click on the button.
    3. Choose the time of the event (B), comment on NDTs (C), and if you are an admin, choose the NDT operator from the dropdown list (D). If you are a basic user, your name is selected by default.
    4. Choose the NDT method(s) and the results for each method (E).
    5. When you have filled in the information, click on the "Log" button (F) to create the new NDT inspection event.

    NDT inspection will be logged (G) into the events section. If you want to log another NDT inspection, you can click on the "Log NDT" button (H) again.

    If any of the NDT inspection methods result in "Rejected" (H), the weld needs to be redone. The next event, therefore, will be "Log weld" (I).

    By following these steps, you can create a new NDT inspection event for your weld. Proper documentation of the welding process is crucial, and logging events such as NDT inspections helps ensure that the highest standards are met. With a clear record of each step of the welding process, it is easy to identify any issues that may arise and address them quickly and efficiently.
