    View document in Documents page

    To view a document on the Documents page of a project, follow these steps:

    1. Navigate to the Documents page for the project.
    2. Expand the section that contains the document you want to view by clicking the arrow icon (A) next to the section name.
    3. Locate the document you want to view and click on the three-dot menu (B) on the bottom right of the document preview.
    4. Select "Open" (C) from the options provided. This will open the document in a new tab of your browser, where you can view or download it.

    By following these steps, you can view any document on the Documents page of your project.
